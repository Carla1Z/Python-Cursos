Aquí voy guardando ejercicios, pruebas de código y pequeños experimentos mientras profundizo en distintos conceptos del lenguaje.

Utilizar este repositorio como espacio de práctica y registro de aprendizaje en Python, organizando el código de manera clara para poder revisarlo y reutilizarlo en el futuro.
